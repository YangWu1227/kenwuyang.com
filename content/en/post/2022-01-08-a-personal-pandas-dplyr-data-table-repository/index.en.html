Motivation

         The content and formatting of this post is heavily inspired by the following comprehensive and awesome post by Atrebas. You could also find him on Twitter here. I use both R and Python extensively in my work everyday, especially the data wrangling packages and library data.table, dplyr (its related siblingz dbplyr, dtplyr etc.), and pandas. I find that, in my case, I often have to translate between different syntax and programmatic thinking when switching from one another. In R, there are ways to accomplishing tasks and there are more Pythonic ways to do things in Python. This is an attempt to create a list of code snippets that can hopefully become a reference, which will be updated over time, for myself and perhaps others as well.

Packages and Library

For R:

library(tibble)
library(dplyr)
library(data.table)

For Python:

import pandas as pd
import numpy as np

Data Frame Creation

Dplyr

tb <- tibble(
  x = c(1, 2, 3),
  y = c(2, 3, 4)
)
tb
## # A tibble: 3 × 2
##       x     y
##   <dbl> <dbl>
## 1     1     2
## 2     2     3
## 3     3     4

Data.table

dt <- data.table(
  x = c(1, 2, 3),
  y = c(2, 3, 4)
)
dt
##    x y
## 1: 1 2
## 2: 2 3
## 3: 3 4

Pandas

df = pd.DataFrame({
  'x': [1, 2, 3], 
  'y': [2, 3, 4]
})
df
##    x  y
## 0  1  2
## 1  2  3
## 2  3  4

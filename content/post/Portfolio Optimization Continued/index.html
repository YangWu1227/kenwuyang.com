       In the previous post, we created a customizable portfolio dashboard via shiny. In that application, the weights of the portfolio are taken as given, and we focus more on the various types of visualizations and performance metrics. In this post, we create a portfolio optimization dashboard, which seeks to find the optimal set of weights that provide a set of optimal portfolios that satisfies some criteria. We will introduce some portfolio theory and seek to implement it in R using the PortfolioAnalytics and tidyquant packages.

       For more readings on the theory, I recommend Essentials of Investments and Practical Portfolio Performance Measurement and Attribution. For implementing these theories in R, I suggest reading the documentations for the aforementioned R packages. Finally, the dashboard can be found here.
